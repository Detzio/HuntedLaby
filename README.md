HuntedLaby
===========

Un jeu d'exploration dans un labyrinthe où le joueur (P) doit atteindre la sortie (case verte) en évitant les chasseurs (H).

Contrôles :
- Z Q S D : déplacer le joueur
- 1 / 2 / 3 : sélectionner la difficulté (1 = facile, 2 = moyen, 3 = difficile)
- R : régénérer le labyrinthe / rejouer
- X : quitter

Notes :
- Implémentation en style fonctionnel (sans var/while/for/foreach).
- Le labyrinthe est généré aléatoirement et garantit un chemin vers la sortie.
